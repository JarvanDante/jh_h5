# 静态资源说明

## 需要的图片资源

请将以下图片放置在 `public` 目录下：

### Logo 和图标

- `logo.png` - 网站 Logo (40x40px)
- `flag-ph.png` - 菲律宾国旗图标 (24x24px)
- `jackpot-icon.png` - Jackpot 图标 (60x60px)
- `jackpot-bg.png` - Jackpot 背景图

### 轮播图

- `banner1.jpg` - 轮播图1 (建议尺寸: 750x360px)
- `banner2.jpg` - 轮播图2
- `banner3.jpg` - 轮播图3

### 游戏封面

- `game1.jpg` - 游戏封面1 (建议尺寸: 375x320px)
- `game2.jpg` - 游戏封面2
- `game3.jpg` - 游戏封面3
- `game4.jpg` - 游戏封面4
- `game5.jpg` - 游戏封面5
- `game6.jpg` - 游戏封面6
